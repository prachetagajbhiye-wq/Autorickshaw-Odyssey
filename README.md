# Autorickshaw Odyssey

## Run Frontend
1. Open `frontend/index.html` in VS Code Live Server or any browser.
2. Mobile-first design supported. Use browser device toolbar for testing.

## Run Backend
1. Navigate to `backend/`
2. Install dependencies: `npm install`
3. Start server: `npm start`
4. Backend runs at `http://localhost:3000`

## Gameplay
- Use buttons to move left/right, boost, or honk.
- Distance, coins, and health update dynamically.
- Procedural city chunks and obstacles to be added in `game.js`.
